== Description ==

[Impressive Slider Made Easy](http://www.wpexpertdeveloper.com/impressive-slider-made-easy/) is an awesome image slider powered by popular open source Jssor jQuery Slider. Provides wide range of transition effects to manage your image sliders.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/impressive-slider-made-easy` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Create new Image Slider using Add New button of Impressive Slider menu item on left menu.
4. Add images to Impressive Slider using WordPress media uploader
5. Customize the settings on Impressive Image Slider - Manage Settings metabox and click Publish button
6. Copy the Impressive Slider shortcode on right side (Ex: [wisme_slider id='2' ]) and add it to a post/page.
7. View the post/page on frontend to see the Impressive Slider
8. Follow the <a href="http://www.wpexpertdeveloper.com/impressive-slider-made-easy/">Documentation and Usage</a>.
